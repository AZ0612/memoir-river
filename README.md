# 记忆河流 · Memory River

> 为长辈打造的回忆录可视化个人网站 —— 用粒子河流动画呈现一生的记忆碎片

## 在线预览

👉 **https://az0612.github.io/memoir-river/**

## 功能特性

- 🌊 **粒子河流动画** — 每段记忆是一颗发光粒子，缓缓流淌
- 🎨 **情绪色彩编码** — 温暖金色、平静蓝色、欢乐粉色...
- 📅 **时间线导航** — 底部年份刻度，悬停高亮对应年份记忆
- 💬 **悬浮摘要** — 鼠标接近粒子时显示一句话记忆摘要
- 📖 **详情弹窗** — 点击展开完整记忆文本、地点、标签、音频
- 🔊 **语音片段** — 内嵌音频播放器，试听回忆者的原声
- 📱 **响应式设计** — 手机/平板/桌面均可访问
- 📄 **纯静态单文件** — 无需服务器，直接打开即可使用

## 快速使用

1. 打开 `index.html`
2. 修改 `MEMOIR_DATA` 中的回忆者信息和记忆数据
3. 部署到任何静态托管服务

## 配套工具

本项目配套 `memoir-river` AI 技能，支持：
- 对话引导（按年代定制的记忆引出问题库）
- 语音录制 & 转写（阿里云百炼 ASR）
- 文字整理 & 修饰（保留口语韵味的文学化整理）
- 网站自动生成（记忆增加，网站自然"生长"）
