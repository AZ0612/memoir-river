# 回忆录对话引导问题库

> 针对 1960s 出生人群的记忆引导，按人生阶段分类。每个问题都是具体、有画面感的，避免空泛。

## 使用原则

1. **一次只问一个问题** — 不要连珠炮
2. **从感官入手** — 气味、声音、触感比"你觉得怎样"更容易唤起记忆
3. **用新闻事件做锚点** — "那年XXX的时候，你在做什么？"
4. **允许沉默** — 沉默是回忆正在浮现的信号
5. **顺着细节深入** — 回忆者提到的任何细节都可以展开

## 一、童年（1961-1975，约 0-14 岁）

### 家与村庄
- 你小时候住的房子是什么样的？有几间屋？地面是泥的还是砖的？
- 夏天的晚上，大人们都在做什么？你会跟着凑热闹吗？
- 你们家附近有没有一棵特别的树、一口井、或者一个水塘？
- 冬天最冷的时候，你怎么取暖的？记不记得脚冻僵的感觉？

### 食物与气味
- 小时候最馋什么东西？得到的时候是什么感觉？
- 你妈妈（或奶奶）最拿手的一道菜是什么？做的时候厨房什么味道？
- 过年能吃到什么平时吃不到的东西？
- 上学带的午饭是什么？有没有和同学换着吃过？

### 游戏与伙伴
- 你小时候最喜欢玩什么？能详细说说怎么玩的吗？
- 有没有一个特别好的小伙伴？你们最常一起做什么？
- 有没有被大人骂的时候？为了什么事？
- 放学回家的路上，一般会经过什么地方？会不会绕远路去哪里玩？

### 学校
- 你的第一位老师长什么样？姓什么？
- 教室里是什么样的桌椅？冬天上课冷不冷？
- 有没有一本书或一个故事让你特别着迷？
- 考试是什么样的？有没有印象特别深的一次？

## 二、少年（1975-1980，约 14-19 岁）

### 时代变迁
- 1976年（毛主席去世）那天你在哪里？周围人什么反应？
- 恢复高考的消息传来时，你是怎么知道的？第一反应是什么？
- 广播里开始播新东西的时候，你最先听到了什么歌或什么节目？

### 第一次
- 第一次离开家去远的地方是什么时候？去哪里？怎么去的？
- 第一次坐火车/汽车是什么感觉？车上的人和味道还记得吗？
- 第一次拿到钱（不是零花钱）是什么时候？用来做了什么？

### 梦想
- 那时候最想成为什么样的人？为什么？
- 有没有一个你特别崇拜或者羡慕的人？

---

## 三、青年（1980-1990，约 19-29 岁）

### 工作
- 你的第一份工作是什么？怎么得到的？
- 上班第一天是什么情形？紧不紧张？
- 同事里面有没有对你特别好的人？或者特别难相处的人？
- 那时候一个月工资多少？第一个月的工资花在哪了？

### 感情
- 你第一次见到妈妈（孩子的母亲）是什么场景？她穿什么？
- 约会都去哪里？那时候流行怎么表达喜欢？
- 结婚那天什么天气？有什么意外或有趣的事？
- 你带回家给她（他）的第一件礼物是什么？

### 生活质感
- 那个年代你家有收音机吗？最喜欢听什么节目？
- 第一次看电视是什么时候？看的什么？
- 你第一次骑自行车是什么时候？自行车什么牌子？

---

## 四、壮年（1990-2010，约 29-49 岁）

### 为人父母
- 你第一次抱到自己孩子的那一刻是什么感觉？
- 孩子小时候做过什么让你又气又好笑的事？
- 你觉得自己和自己父母带孩子最大的不同是什么？

### 事业
- 工作上最骄傲的一件事是什么？
- 有没有做过一个很艰难的选择？当时怎么想的？
- 如果时间倒流，有没有什么事你会做不同的选择？

### 时代浪潮
- 1997年香港回归那天你在做什么？
- 2008年奥运会开幕式你在哪里看的？什么感受？
- 第一次用手机是什么时候？什么牌子？
- 第一次上网是什么感觉？看了什么？

---

## 五、此刻（2010-至今，约 49 岁以后）

### 回望
- 如果用一个词形容你的前半生，你会选什么词？
- 你觉得你从父亲/母亲那里继承了什么？
- 有没有一个时刻你觉得"我这辈子值了"？
- 你最想对年轻时候的自己说一句什么话？

### 传承
- 你希望后代记住你什么？
- 有没有一句话或者一个道理是你一直信奉的？
- 你觉得现在的年轻人最缺什么？最不缺什么？

## 新闻锚点年表（1961-2025）

用于勾连个人记忆和历史事件：

| 年份 | 事件 | 引导语示例 |

| 1966 | 文革开始 | "那年你刚上小学吧，学校有变化吗？" |
| 1969 | 人类登月 | "不知道那时候你听没听说过这事儿？" |
| 1976 | 唐山大地震/主席去世 | "76年那几件大事，你还记得细节吗？" |
| 1977 | 恢复高考 | "高考恢复了，你身边有人去考了吗？" |
| 1978 | 改革开放 | "开始'分田到户'了，你家分了多少地？" |
| 1984 | 国庆阅兵/洛杉矶奥运 | "许海峰拿金牌那天你在哪？" |
| 1987 | 第一部大哥大 | "那时候见没见过'大砖头'手机？" |
| 1992 | 南巡讲话 | "下海潮的时候你动没动过心？" |
| 1997 | 香港回归 | "7月1号那天你在做什么？" |
| 1998 | 大洪水 | "98年发大水，你那边受影响了吗？" |
| 2001 | 申奥成功/中国入世 | "那天晚上你看转播了吗？" |
| 2003 | 非典 | "非典那阵子你是怎么过的？" |
| 2008 | 奥运/汶川地震 | "5月12号那天你在哪里？" |
| 2020 | 疫情封控 | "封城那段日子你每天怎么过的？" |

## 对话结尾仪式

每次对话结束时，用一句温暖的话收尾：

- "谢谢你把这些告诉我，这些记忆很珍贵。"
- "你讲的这些画面，我都能看到。下次我们接着聊。"
- "你的故事值得被记住。我们下次继续。"
